Politique de Confidentialité

Vos données sont protégées.

Dernière mise à jour : Avril 2026

1. Collecte de données

L'application Compagnon Astrelya ne collecte, ne stocke, ni ne transmet aucune donnée personnelle à des serveurs tiers. Tous les calculs sont effectués localement sur votre appareil.

2. Authentification Microsoft Entra ID

L'application utilise Microsoft Entra ID (Azure Active Directory) pour l'authentification. Ce processus est géré directement par Microsoft et soumis à la politique de confidentialité de Microsoft. Astrelya ne stocke aucun identifiant ni mot de passe de votre compte Microsoft.

3. Intégration Google Assistant / Gemini

L'application peut être invoquée par Google Assistant ou Gemini via des App Actions Android. Lorsque cela se produit :

  • Les paramètres de calcul (TJM, salaire) sont transmis localement par Android
  • Aucun de ces paramètres n'est envoyé à des serveurs extérieurs
  • Le résultat est calculé sur l'appareil et prononcé vocalement

4. Permissions Android

L'application utilise les permissions suivantes :

  • INTERNET : Uniquement pour l'authentification Microsoft Entra ID

5. Conservation des données

Aucune donnée saisie dans l'application (salaires, TJM, marges) n'est conservée entre les sessions ni synchronisée en ligne.

6. Contact

Pour toute question concernant la protection des données, contactez l'éditeur de l'application : contact@astrelya.com
